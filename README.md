### 适用范围

本标准适用于以下类型的电动汽车：

- 纯电动汽车
- 插电式混合动力电动汽车
- 燃料电池电动汽车

### 主要内容

GB/T 40855-2021 主要涵盖了以下内容：

1. **信息安全要求**：详细规定了电动汽车远程服务与管理系统在信息安全方面的具体要求。
